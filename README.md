To run the website:

1. Navigate to eaihub folder
2. Create a virtual environment -
   python3 -m venv .venv
3. Activate the environment -
   . .venv/bin/activate or source .venv/bin/activate
4. Install the requirements -
   pip install -r requirements.txt
5. Run the app -
   python app.py
